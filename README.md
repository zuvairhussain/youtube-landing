# YouTube Landing Page

A modern, responsive landing page designed to encourage visitors to subscribe to your YouTube channel. Built with HTML, CSS, and JavaScript for a smooth, engaging user experience.

## 🌟 Features

- **Modern Design**: Clean and professional UI with gradient backgrounds
- **Fully Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth animations, hover effects, and scroll animations
- **Call-to-Action**: Multiple conversion points to encourage subscriptions
- **Video Integration**: Embedded YouTube video showcase
- **Fast Performance**: Optimized for quick loading times
